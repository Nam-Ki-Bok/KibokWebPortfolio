Falling Grade

개요

- 총알 피하기, 리듬게임 등 고전 게임의 개념을 응용.

- 피하기만 하는 게임이 아닌 복합적인 게임을 구성하려고 노력.

- 우리에게 친근하게 다가올 수 있는 주제를 선택.

- 학점이 떨어진다면 상대적으로 낮은 학점은 피하려고 할 것이고 높은 학점은 얻으려고 할 것이다.

사용 언어 : JAVA
개발 도구 : Eclipse

임시 게임 디자인

Adobe XD를 이용한 세부 디자인 계획

- 각각의 page에 어울리는 화면과 버튼 완성

- 떨어지는 학점은 구분 가능하게 각각 다른 색으로 계획

- GameOver 화면과 Clear 화면은 다르게 디자인

- 모든 버튼은 마우스가 올라갔을 경우 이미지가 바뀌게 계획

- 기존의 메뉴바 대신 JLabel을 이용, 메뉴바와 종료버튼 디자인

게임 화면 구성

- 게임에 필요한 모든 이미지, 버튼 생성.

- 각각의 화면을 표시할 변수 설정.

- if문, 변수, setVisible 메소드를 통해 상황에 맞는 버튼, 이미지 출력

게임 결과 산출

- static 변수를 통한 게임 결과 계산.

- 캐릭터와 학점이 접촉 시 학점의 생성값으로 점수 부여.

- double 값으로 계산 후 String.format 메소드를 통해 출력.

게임 구동 원리

- 난이도 별 학점 생성 빈도 수 조절.

- 랜덤 생성 값에 따른 학점 종류 조절.

- 랜덤 하강 값에 따른 학점 하강 속도 조절.

- 캐릭터 이미지와 학점 이미지 좌표 계산을 통한 점수 계산.

- KeyAdapter 클래스를 이용한 캐릭터 이동.
