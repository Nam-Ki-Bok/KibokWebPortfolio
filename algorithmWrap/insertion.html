- 배열의 앞에서부터 정렬된 부분과 비교해 자신의 위치를 찾아 삽입하는 정렬

- 왼쪽의 그림과 같이 Key 값 기준 왼쪽의 수들을 비교해 위치를 찾으면 된다

- 아래의 코드는 오름차순 코드이다, 만약 내림차순으로 하고 싶다면 if문의 부등호만 반대로 바꾸면 된다

- for문 안쪽 코드를 한줄로 줄일 수 도 있는데 잘 생각해보자 !

void insertionSort(int *arr, int arrSize) {
    int i, j, key;
    
    // 배열의 원소 개수 - 1 만큼 반복한다.
    for(i = 1; i < arrSize; i++) {
        key = arr[i];
        for(j = i - 1; j >= 0; j--) {
            if(key < arr[j]) {
                arr[j+1] = arr[j]; // key 값 보다 크다면 오른쪽으로 한칸 이동
            
            else
                break;
        }
        arr[j+1] = key; // 비교 후 j 값은 key 값이 삽입될 위치 - 1 이 되므로

    }
}
