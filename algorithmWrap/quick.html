- merge sort와 마찬가지로 분할정렬 알고리즘 중 하나이다

- pivot 값을 정해 pivot 보다 작은 값은 왼쪽

   pivot 보다 큰 값은 오른쪽으로 정렬한다

- 이미 정렬 된 데이터에 사용할 경우 최악의 시간 복잡도를 가지게 된다

- 밑에 코드와 함께 보면 이해가 가겠지만

   우선 맨 왼쪽 값을 pivot으로 둔다

   그 후 i를 증가시키면서 pivot과 비교한다

   i와 j의 값이 다르면서

   pivot 값이 i 값 보다 클 경우

   j++ 한 뒤 i와 j를 swap 한다

   반복을 마친 후 pivot 값과 j 값을 swap 한다

   그러면 pivot 값은 고정 값이 된다

   또 pivot 기준으로 왼쪽은 pivot 보다 작은 값

   오른쪽은 pivot 보다 큰 값이 오게 된다

void quickSort (int *arr, int left, int right) {
   
    int pivot, i, j, tmp;
    pivot = left;
    j = pivot;
    
    if (left < right) {
        for (i = pivot + 1; i <= right; i++) {
            if (arr[pivot] > arr[i]) {
                j++;
                if (i != j) {
                    tmp = arr[i];
                    arr[i] = arr[j];
                    arr[j] = tmp;
                }
            }
        }
        
        // swap을 하고 나면 pivot을 기준으로
        // 작은 값은 왼쪽, 큰 값은 오른쪽으로 위치한다
        tmp = arr[pivot];
        arr[pivot] = arr[j];
        arr[j] = tmp;
        
        // 배열을 분할 할 수 있을 때 까지 분할한다
        pivot = j;
        quickSort(arr, left, j-1);
        quickSort(arr, j+1, right);
    }
}
